example-advanced
================

An advanced example package demonstrating ``sphinxcontrib-pydantic`` with
complex Pydantic patterns.

This example shows:

- **Model inheritance hierarchies** - Base classes with validators
- **Nested/composed models** - Configuration objects with nested configs
- **Computed fields** - Fields derived from other fields
- **SQLModel tables and DTOs** - Database models with relationships
- **BaseSettings** - Environment-based configuration
- **Generic models** - Type-parameterized reusable patterns
- **Field validators** - All validation modes (before, after, wrap)
- **Model validators** - Cross-field validation
