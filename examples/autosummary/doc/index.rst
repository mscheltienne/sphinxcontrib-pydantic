example-autosummary
===================

This example demonstrates how to document Pydantic models and SQLModel tables
using ``autosummary`` with custom templates.

Key features demonstrated:

- Using ``.. autosummary::`` with custom templates for Pydantic models
- Organizing models into logical sections (configuration, database tables, DTOs)
- Proper ``numpydoc`` configuration for Pydantic models
- Handling SQLModel inheritance
